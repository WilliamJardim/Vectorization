# Root
Nesta página você pode encontrar informações sobre a classe **VectorizationJS.Root**, uma classe interna do VectorizationJS que serve como base para todas as demais classes como **VectorizationJS.Matrix**, **VectorizationJS.Vector** e **VectorizationJS.Scalar**.

# Utilidade
Ela tem vários metódos para uso interno do VectorizationJS, como funções que fazem tradução de nome de métodos, atributos, inicializações de paramêtros globais que qualquer classe vai precisar usar, etc.
